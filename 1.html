HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browers reveive HTML documents from a web server or from loacal storage and render them into multimedia web pages. HTML describes the strcture of a web page semantically and orginally included cues for the apperance of the document.

HTML elements are the building blocks of HYML pages. With HYML contruct, images and other object, such as interactive forms, may be enbedded into the rendered page. It provides a means to creat structured documents by denoting structural semantics for text such as headings, parahraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
